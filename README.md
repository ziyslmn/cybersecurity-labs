# Cybersecurity Labs

Hands-on cybersecurity and digital forensics investigations completed during my Cybersecurity program at Northeastern Illinois University.

These projects demonstrate practical experience in:

• malware analysis  
• Windows and Linux forensic investigations  
• disk image analysis  
• volatile artifact collection  
• file recovery and data carving
---

## Featured Projects

### [Static Malware Analysis Investigation](./static-malware-analysis)
Performed static analysis on a suspicious Windows executable to identify indicators of compromise, analyze PE structure, and investigate potential command-and-control behavior.

### [Windows System Forensics Investigation](./windows-forensics-investigation)
Investigated Windows artifacts including event logs, system activity, and user behavior to reconstruct system events and identify security-relevant activity.

### [Linux System Forensics Investigation](./linux-forensics-investigation)
Analyzed Linux system artifacts, network activity, and potential rootkit indicators using forensic and network analysis tools.

### [Volatile Data Collection and Live System Analysis](./volatile-data-collection)
Collected and analyzed volatile artifacts from a running system including processes, network connections, routing tables, and system memory information.

### [Disk Image Forensics (Autopsy Investigation)](./disk-image-forensics-autopsy)
Used Autopsy to analyze a forensic disk image, recover artifacts, examine file system structures, and reconstruct potential user activity.

### [File Recovery and Data Carving](./file-recovery-data-carving)
Recovered deleted files and artifacts from storage media using forensic data carving tools and disk imaging techniques.
