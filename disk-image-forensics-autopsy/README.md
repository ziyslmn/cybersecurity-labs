# Disk Image Forensics Investigation (Autopsy)

## Objective
The objective of this investigation was to analyze a forensic disk image using Autopsy to identify system artifacts, recover files, and reconstruct user activity.

## Tools Used
• Autopsy  
• Sleuth Kit  

## Investigation Activities

The investigation involved examining a forensic image of a USB device using Autopsy. The following forensic tasks were performed:

- Importing and analyzing a forensic disk image
- Examining file system structures
- Identifying existing and deleted files
- Reviewing metadata and timestamps
- Investigating potential user activity artifacts

## Findings

The analysis demonstrated how forensic investigators can reconstruct system activity by analyzing disk images. File metadata, timestamps, and recovered artifacts provide important evidence when investigating digital systems.

## Skills Demonstrated

• Disk image forensic analysis  
• File system artifact investigation  
• Evidence recovery  
• Digital investigation documentation
