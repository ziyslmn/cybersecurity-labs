# Disk Image Forensics Investigation (Autopsy)

## Objective
The objective of this investigation was to analyze a forensic disk image using Autopsy in order to identify file system artifacts, recover files, and reconstruct system activity.

## Tools Used

- Autopsy
- Sleuth Kit

## Investigation Workflow

1. Import forensic disk image into Autopsy.
2. Examine file system structure.
3. Identify existing and deleted files.
4. Review file metadata and timestamps.
5. Analyze recovered artifacts.
6. Correlate artifacts to reconstruct system activity.

## Findings

The analysis demonstrated how forensic investigators can reconstruct user activity and system events using disk image artifacts and file metadata.

## Skills Demonstrated

- Disk image forensic analysis
- File system investigation
- Artifact recovery
- Digital evidence documentation
