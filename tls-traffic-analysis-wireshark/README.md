# TLS Traffic Analysis with Wireshark

This project investigates a Transport Layer Security (TLS) session using Wireshark to understand how secure HTTPS connections are established between a client and a web server.

The investigation analyzes the TCP connection establishment, TLS handshake process, certificate exchange, and encrypted application data.

Skills Demonstrated

• Network traffic analysis  
• TLS handshake investigation  
• Packet inspection using Wireshark  
• HTTPS protocol analysis  

Tools Used

Wireshark  
TLS protocol analysis  
Packet inspection
