# File Recovery and Data Carving Investigation

## Objective
The objective of this investigation was to recover deleted files from a storage device using forensic data carving techniques.

## Tools Used

• recoverjpeg  
• Foremost  
• bulk_extractor  
• dcfldd  

## Investigation Activities

The investigation included several forensic techniques used to recover deleted data from a disk image:

- Creating a forensic disk image using dcfldd
- Recovering deleted image files using recoverjpeg
- Performing file carving using Foremost
- Extracting artifacts such as URLs and metadata using bulk_extractor

## Findings

The analysis demonstrated how forensic tools can recover deleted files and extract artifacts from storage media. Data carving techniques allow investigators to recover files even when file system metadata has been removed.

## Skills Demonstrated

• File recovery and data carving  
• Forensic disk imaging  
• Artifact extraction  
• Digital evidence analysis
