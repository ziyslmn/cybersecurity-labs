# File Recovery and Data Carving Investigation

## Objective
The objective of this investigation was to recover deleted files and artifacts from a storage device using forensic data carving techniques.

## Tools Used

- recoverjpeg
- Foremost
- bulk_extractor
- dcfldd

## Investigation Workflow

1. Create a forensic disk image using dcfldd.
2. Perform file recovery using recoverjpeg.
3. Perform file carving using Foremost.
4. Extract artifacts using bulk_extractor.
5. Analyze recovered files and metadata.
6. Document recovered evidence.

## Findings

The investigation demonstrated how deleted files can be recovered from storage media even when file system metadata has been removed.

## Skills Demonstrated

- File recovery
- Data carving
- Forensic disk imaging
- Digital artifact analysis
